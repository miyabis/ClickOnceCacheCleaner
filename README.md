# ClickOnce Cache Cleaner

[System.Deployment.Application.DeploymentServiceCom.CleanOnlineAppCache](https://docs.microsoft.com/ja-jp/dotnet/api/system.deployment.application.deploymentservicecom.cleanonlineappcache?view=netframework-4.0) を使ってインストールされているオンライン専用の ClickOnce アプリケーションをすべて削除します。
